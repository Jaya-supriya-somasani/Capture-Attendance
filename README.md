# Capture-Attendance
📍 Android attendance app built with Jetpack Compose — captures photo, records live GPS location, reverse geocodes address, and stores all records locally using Room Database.

✨ Features
📷 Camera Capture — Launch the device camera and capture attendance photo with one tap
📍 Live Location — Automatically fetches GPS coordinates after every capture
🏙️ Reverse Geocoding — Converts latitude/longitude into a readable address (city, area, state)
🗃️ Local Storage — All records saved persistently using Room Database
📜 Attendance History — View all past captures with photo, address, and timestamp
🔐 Runtime Permissions — Handles Camera and Location permissions gracefully
⚡ Optimized UI — Built with single-state recomposition to prevent UI flickering
